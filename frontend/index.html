MVP для дипломного проекта

Анализ отзывов о товарах

Система собирает отзывы из CSV-файлов и API маркетплейсов, очищает текст, определяет общую тональность, находит проблемные аспекты и формирует краткие бизнес-рекомендации. Все результаты собраны в одном дашборде: можно отслеживать настроение покупателей, частые причины негатива и изменения аналитики после удаления нерелевантных отзывов.
